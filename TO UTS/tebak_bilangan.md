# C. Menebak Bilangan

|Time limit|	1 s |
|:---|:---|
|**Memory limit**|	**128 MB** |
 
cutie_panda kembali ditantang oleh teman online-nya untuk menyelesaikan masalah berikut!

Diberikan sebuah bilangan `n`, cutie_panda diminta untuk mencari sebuah bilangan bulat `x` sehingga:
- `2 ≤ x ≤ n`
- `x + 2x + 3x + ... + kx` dimana `kx ≤ n` adalah maksimum.

Bantulah cutie_panda untuk menemukan bilangan bulat x!

## Input
Baris pertama berisi sebuah bilangan 
- `t (1 ≤ t ≤ 100)`, menandakan banyak kasus uji.
- t baris berikutnya berisi sebuah bilangan bulat `n (2 ≤ n ≤ 1000)`.

## Output
Untuk setiap kasus uji, keluarkan sebuah bilangan bulat x pada setiap barisnya!

## Contoh Masukan 1
```
2
7
12
```

## Contoh Keluaran 1
```
2
2
```

## Contoh Masukan 2
```
1
2026
```

## Contoh Keluaran 2
```
2
```
