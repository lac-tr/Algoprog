## A. Total Belanja

Time limit	`500 ms`

Memory limit	`256 MB`

Description
-
Sebuah toko menjual satu jenis barang dengan harga tertentu, misalkan *p* rupiah. Pembeli dapat membeli barang tersebut. Setiap pembelian dikenakan pajak 10% dari total nilainya. Buat program C untuk menghitung dan menampilkan total harga yang harus dibayar jika dilakukan pembelian sebanyak 
*n* buah.

Constraints
-
- 1000 ≤ `p` ≤ 2000000
- 1 ≤ `n` ≤ 1000

Input
-
Input data diberikan dengan format:
``
p n
``

Output
-
Total harga yang harus dibayar setelah dikenakan pajak 10%. Nilai output ditampilkan dengan 2 digit di belakang tanda desimal.

Sample Input
-
    5000 10

Sample Output
-
    55000.00
